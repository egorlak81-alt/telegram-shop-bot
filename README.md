# 🛍 Telegram Shop Bot

Telegram-бот магазина одежды с веб-приложением (Web App) и Google Таблицей в качестве базы данных.

![Telegram](https://img.shields.io/badge/Telegram-Bot-blue?logo=telegram)
![Google Apps Script](https://img.shields.io/badge/Google-Apps%20Script-green?logo=google)

## Возможности

- 📱 Веб-приложение (Telegram Web App) с каталогом товаров
- 🔍 Поиск по названию и описанию
- 🏷 Фильтрация по типу товара
- 💰 Сортировка по цене (по возрастанию/убыванию)
- 📋 Копирование информации о товаре
- 💬 Кнопка для перехода в ЛС продавца
- 📊 Google Таблица как база данных — без сервера
- 🎨 Адаптивный дизайн, тёмная тема, плавные анимации
- 📲 Автоматическая подстройка под тему Telegram

## Архитектура

```
Google Таблица (база данных)
       ↕
Google Apps Script (бэкенд + API)
       ↕
Telegram Bot ←→ Web App (HTML)
```

## Быстрый старт

### 1. Создай бота
- Открой [@BotFather](https://t.me/BotFather) в Telegram
- Отправь `/newbot`, получи **токен**
- Узнай свой Chat ID через [@userinfobot](https://t.me/userinfobot)

### 2. Создай Google Таблицу
Создай таблицу с листом **«Товары»** и заголовками:

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| ID | Название | Фото | Тип | Описание | Цена |

### 3. Настрой Apps Script
1. В таблице: **Расширения → Apps Script**
2. Создай два файла: `Code.gs` и `shop.html`
3. Скопируй код из этого репозитория
4. Заполни переменные в `Code.gs`:

```javascript
const BOT_TOKEN     = 'твой_токен';
const OWNER_CHAT_ID = 'твой_chat_id';
const OWNER_USERNAME = 'твой_юзернейм';
const DEPLOYED_URL  = ''; // заполнишь после деплоя
```

### 4. Деплой
1. **Развертывание → Новое развертывание**
2. Тип: **Веб-приложение**
3. Доступ: **Все**
4. Скопируй URL, вставь в `DEPLOYED_URL`
5. Передеплой с **новой версией**

### 5. Вебхук
1. В Apps Script выбери функцию `setWebhook`
2. Нажми **Выполнить**

### 6. Готово!
Открой бота, нажми `/start` → **Открыть магазин**

## Структура таблицы

| Столбец | Содержимое | Пример |
|---------|-----------|--------|
| A — ID | Порядковый номер | 1 |
| B — Название | Название товара | Худи Classic Black |
| C — Фото | Прямая ссылка на фото | https://i.ibb.co/xxx/photo.jpg |
| D — Тип | Категория | Худи |
| E — Описание | Краткое описание | Чёрное худи оверсайз |
| F — Цена | Цена с валютой | 89.90р |

## Загрузка фото

1. Загрузи фото на [imgbb.com](https://imgbb.com)
2. Открой ссылку на фото в браузере
3. ПКМ на фото → **Копировать адрес изображения**
4. Вставь ссылку (должна начинаться с `i.ibb.co`)
